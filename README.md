Это не просто CRUD-приложение.  
Это **рабочий инструмент сисадмина**, который я использую каждый день в реальной локальной сети предприятия.

## Что умеет проект прямо сейчас

| Функция | Как работает | Зачем это админу |
|--------|--------------|------------------|
| **Учёт сотрудников, компьютеров, принтеров, телефонов, учёток** | Классический ресурсный CRUD на современном Laravel 11 + Blade + Tailwind | Всё в одном месте, без Excel-таблиц |
| **Мониторинг Wi-Fi точек доступа** | Прямая ссылка на Omada SDN Controller (`#devices/ap`) в новой вкладке | Три зелёные точки — всегда под контролем |
учётные записи сервисов используемых на предприятии зашифрованы, доступны через ввод мастер-пароля ( для безопасности). 
 мониторинг в реальном времени компьютеров, принтеров. реализован телеграмм бот с уведомлениями при критичном состоянии заправки картриджей на принтерах.
 **Поиск человека по ДРФО** | Вводишь 10 цифр → Enter → мгновенно выскакивает модалка с ФИО, датой рождения, адресом, квартирой, телефоном и местом работы | За 3 секунды понимаешь, кто перед тобой |
| **Интеграция с базой ДРФО** | Прямой API-запрос к старому Laravel 5 проекту (`192.168.13.111/api/drfo-search`) — мы это победили через CORS, header() и 500-е ошибки :) | Две отдельные системы стали одной |
| **Адаптивный интерфейс** | Tailwind CSS + Alpine.js — красиво на любом экране | Можно работать с телефона в серверной |

## Технологии

- Laravel 11
- Blade + Tailwind CSS
- Alpine.js (лёгкий интерактив)
- MySQL
- Внутреннее API (Laravel 5 → Laravel 11)
- Fetch API + модальные окна

## Как запустить у себя (локально или в сети)

```bash
git clone https://github.com/nickon72/sysadmin-app.git
cd sysadmin-app
composer install
cp .env.example .env
php artisan key:generate
# настрой .env (база данных)
php artisan migrate --seed
php artisan serve
# или настраивай под Laragon/OpenServer/Nginx
